Вот полный список всего, что использовано из Bootstrap 5 / MDB, с разбивкой по категориям:

---

### Сетка (Grid)
| Класс | Где |
|---|---|
| `container-fluid` | галерея, контент |
| `row` | галерея, карточки, сайдбар-айтемы |
| `row g-0` | галерея (без отступов между колонками) |
| `row g-2` | карточки (маленький gap) |
| `col-12` | все колонки на мобильном |
| `col-md-6` | карточки — 2 в ряд от md |
| `col-lg-4` | правая галерея, сайдбар — от lg |
| `col-lg-8` | левая галерея, блок карточек — от lg |
| `col-5` / `col-7` | колонки внутри сайдбар-айтема |

---

### Navbar
| Класс | Где |
|---|---|
| `navbar` | `<nav>` |
| `navbar-brand` | логотип |

---

### Карточки (Cards)
| Класс | Где |
|---|---|
| `card` | блок карточки |
| `card-body` | тело карточки |

---

### Кнопки
| Класс | Где |
|---|---|
| `btn` | кнопки "Подробнее" в сайдбаре |
| `btn-primary` | синий цвет кнопки |
| `btn-sm` | маленький размер кнопки |

---

### MDB — Hover-эффекты
| Класс | Где |
|---|---|
| `bg-image` | обёртка над каждой картинкой в галерее и карточках |
| `hover-zoom` | zoom при наведении (из mdbootstrap.com/docs/standard/content-styles/hover-effects/) |

---

### Утилиты — Flexbox
| Класс | Где |
|---|---|
| `d-flex` | navbar, card-body |
| `d-block` | кнопки сайдбара |
| `flex-wrap` | ссылки в navbar |
| `flex-column` | card-body (вертикальный flex) |
| `flex-grow-1` | параграф в карточке (занимает всё место) |
| `align-items-center` | сайдбар-айтемы |

---

### Утилиты — Spacing
| Класс | Где |
|---|---|
| `px-3`, `px-5` | горизонтальные padding |
| `py-0` | navbar без вертикального padding |
| `ps-3` | padding-start у сайдбара |
| `mt-1`, `mt-2`, `mt-3`, `mt-4` | margin-top |
| `mt-lg-0` | сброс margin-top от lg |
| `mb-0`, `mb-2`, `mb-4` | margin-bottom |
| `ms-auto` | кнопка к правому краю |
| `p-0`, `p-3` | padding у card-body / card |

---

### Утилиты — Размеры
| Класс | Где |
|---|---|
| `w-100` | картинки галереи — 100% ширины |
| `h-100` | карточки одинаковой высоты |
| `img-fluid` | картинки в сайдбаре |

---

### Утилиты — Текст / Типографика
| Класс | Где |
|---|---|
| `fw-bold` | заголовки карточек |
| `fs-5` | размер шрифта заголовков |
| `small` | уменьшенный текст параграфов |
| `text-center` | заголовок карточки |
| `text-end` | "Подробнее" справа, текст сайдбара |
| `text-secondary` | серый цвет текста параграфов |
| `text-muted` | футер |

---

### Утилиты — Borders / Corners
| Класс | Где |
|---|---|
| `border` | рамка карточки |
| `border-start` | левая граница сайдбара |
| `rounded-2` | картинки сайдбара |
| `rounded-4` | карточки, картинки в карточках |

---

### Утилиты — Nav
| Класс | Где |
|---|---|
| `nav-link` | ссылки в navbar |
| `active` | активная ссылка |
